Opening Answer Dialog

First add a button to your stack then right click on button and click Edit script. You will see a new window and inside that window enter this code (just copy and paste)




on mouseup

answer "Here you can enter text that you want to display"

end mouseup

Showing text form field in answer dialog


Add a field and a button to your stack. There are 2 ways to do this so i am going to show you boath ways

First way

on mouseUp

answer field "Field"

end mouseup

Second way

on mouseUp

put field "Field" into blah

answer blah

end mouseup

Let's talk about command "on mouseUp"

Actually you don't have to put on mouseUp you can put on mouseDown or on mouseDoubleup and so on...

on mouseUp will do run the code of the object after you RELEASE a mouse click

on mouseDown will do run the code of the object after you PUSH a mouse click

on mouseDoubleup will do run the code of the object after you DOUBLE CLICK a mouse click
